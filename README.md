## C
This project is a tiny C interpreter.

#### Fetures
- [x] decimal, hex, char and string literal
- [x] arith `+, -, *, /, %`
- [x] relational `>, <, >=, <=, !=, ==`,
- [x] unary `+, -, !`
- [x] assignment `=, +=, -=`
- [ ] increment `++, --`
- [x] logical `&&, ||`
- [ ] bitwise `&, |, >>, <<`
- [ ] address `*, []`
- [ ] dereference `&`

- [x] branch `if, else, ?:`
- [x] iteration `do, while`

- [ ] `enum`
