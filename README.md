## C
This project is a tiny C interpreter.

#### Fetures
- [x] decimal, hex, char and string literal
- [x] arith `+, -, *, /, %`
- [x] relational `>, <, >=, <=, !=, ==`,
- [ ] unary `+, -`
- [x] assignment `=, +=, -=`
- [ ] increment `++, --`
- [ ] logical `&&, ||, !`
- [ ] bitwise `&, |, >>, <<`
- [ ] address `*, []`
- [ ] dereference `&`

- [ ] branch `if, else, ?:`
- [ ] iteration `do, while, for, break, continue`

- [ ] `enum`
