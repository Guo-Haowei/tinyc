## C

- [x] arith `+, -, *, /, %`

- [x] assignment `=, +=, -=`
- [x] increment `++, --`
- [x] logical `&&, ||`
- [x] bitwise `&, |, >>, <<`
- [x] address `*, []`

- [x] branch `if, else, ?:`
- [x] iteration `while, break, continue`
